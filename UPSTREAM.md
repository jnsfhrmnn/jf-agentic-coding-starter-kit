## Thank You

This project would not exist without Alex Sprogis and the original AI Coding
Starter Kit. The upstream project supplied the core idea of guiding product work
through reusable AI coding skills. Thank you, Alex, for publishing that foundation
for the community.

This derivative takes a different, complementary direction: it removes the fixed
web stack, supports Claude Code and Codex side by side, keeps all skills local to
the repository, adds tracked adoption of existing projects, and introduces durable
task, review, workflow-state, and branch-finish contracts.
