## Status Legend
- **Roadmap** - `/1-csk-init` done, feature identified in feature map, no spec file yet
- **Planned** - `/2-csk-write-spec` done, full spec written, architecture not yet designed
- **Architected** - `/3-csk-architecture` done, tech design approved, ready to build
- **In Progress** - `/4-csk-frontend` or `/5-csk-backend` active or completed, not yet in QA
- **In Review** - `/6-csk-qa` active, testing in progress
- **Approved** - `/6-csk-qa` passed with recorded evidence, no critical/high bugs, ready for the approved release/package/deploy target
- **Deployed** - `/7-csk-deploy` done with release evidence, released, packaged, deployed, or otherwise handed off through the approved target
- **Cancelled** - intentionally closed with a one-line reason in the spec; terminal status, no next command
